Infotext Demo & Guide

Summary

This is a CSS only experiment for showing a tooltip that can be customised through classes.

To use, add the data-infotext attribute to an element with the value of your tooltip.

The component works by adding in styles to an element's pseudo elements (::before and ::after). These elements aren't supported by input and img elements, so the tooltip won't appear when added to these.

Examples

Default

The default behaviour is to display the tooltip on the top-left of the element

<div data-infotext="Basic Example">Basic Infotext example</div>
Basic Infotext example

Positioning

By adding the classes infotext-right and/or infotext-bottom the position of the tooltip is changed (relative to the element it belongs to)

<button data-infotext="Tooltip on bottom-right" class="infotext-right infotext-bottom">Positioning tooltip on bottom-right</button>

Alert and Inverse

The tooltip can be used for form validation by combining the classes infotext-alert and infotext-inverse to show error messages that disappear when the user mouses over the content

<div data-infotext="Please enter your name here" class="infotext-inverse infotext-alert infotext-center infotext-bottom infotext-text-center">
  <label for="example-name">Your Name: </label>
  <input type="text" id="example-name">
</div>

Wider Tooltip

The CSS isn't aware of the amount of content being displayed, so it can't determine by itself the right width for the tooltip. You can add the infotext-wide (or conversely infotext-narrow) to display more appropriate widths for the content you need to display.

<a href="#" data-infotext="This is a wider tooltip for when more text needs to be displayed. It's 400px wide by default, but this can be adjusted in the SCSS file." class="infotext-wide infotext-bottom infotext-right">Help</a>
Help

Animated Tooltip

It's possible to add in the infotext-slide-up or infotext-slide-down classes to add in a simple animation along with the fade in/out.

<a href="#" data-infotext="Don't worry, help is here!" class="infotext-bottom infotext-right infotext-slide-up infotext-narrow">
Help

Persistant Tooltip

If the tooltip is to remain on screen permanently, use the infotext-persist class. You can also combine this with the animation classes for a subtle interaction

<button class="infotext-persist infotext-bottom infotext-slide-down infotext-text-right" data-infotext="Click this button to submit the form and enter the competition">Submit Form</button>

Quick Guide

Basic Usage
Add the attribute data-infotext to any HTML element
Position options
Use the classes infotext-top or infotext-bottom, and infotext-left and infotext-right to position the tooltip against the element it appears on
Alerts
Use the infotext-alert class to style the tooltip for urgent messages
Persistant Tooltips
For tooltips that don't rely on user events (they appear by default) use the infotext-persist class
Inverse Action
If you need a tooltip that appears by default, but disappears on hover, use the infotext-inverse class
Text Alignment
If you need to align the text to the center or right, use the classes infotext-text-center and infotext-text-right classes
Wider Tooltip
By default, the tooltip is 250px wide (this can be configured in the SCSS file). To make the tooltip wider (400px), use the class infotip-wide
Narrower Tooltip
Change the width of the tooltip to 100px. This can be reconfigured in the SCSS file
Animated Tooltip
Add the class infotext-slide-down or infotext-slide-up for a simple animated effect
